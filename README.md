# Winning in Major League Baseball & the Statistics that Drive It

## Overview
Using MLB game data aggregated to the season level for each team, create a linear regression model that will allow for determining which statistics contribute most towards a team's win totals.

## Problem
Functioning as a data analytics team working within the front office of an MLB team, use analytics to create an edge for evaluating players during scouting efforts.

## Approach
Create a linear regression model that uses a historical season level data for each team to predict their win totals and provide the ability to interpret the weight of each feature.

## Results
Stats that resulted from runners advancing around the base path were found to contribute most heavily to wins. This helped identify that the archetype of players that our team wants to target are speedsters that won't hesitate to take an extra base, which will be used for future scouting, drafting, and player development efforts.
